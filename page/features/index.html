Features

  • 100% JSON driven UI allows quick and simple copy/paste document operations.
  • Connect to any number different datbases in one UI. Even different types of databases.
  • Documents are validated against a schema ( json-schema.org ) before saving database. Never again accidentally corrupt random documents in your collections with missing and/or invalid attribs.
  • Create any number of different collection queries and run with 1 click
  • Easily write complex Firestore queries including OR operations
  • Export any query as a complete/runable JavaScript cloud function
  • Schema and Queries JSON docs can include both line and block commenting
  • Queries can include 'input points' that will prompt you to enter variable data when the query runs
  • Write 'repair' queries that can add missing attribs and change attrib values on conditional sets of collection docs.
  • Queries automatically honor soft delete attribs.
  • Write 'purge' queries that will hard delete conditional sets of soft deleted collection docs.
  • Copy/paste collection and sub-collection data between many open tabs.
  • Simple/intuitive sub-collection references in queries.
  • Sort asc/desc on up to 4 attribs without needing to create indexes.
